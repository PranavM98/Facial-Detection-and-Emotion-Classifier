Physical Layer Data Augmentation Techniques for
Face Recognition and Face Emotion Classification Data

Paper PDF


We built an application where the user can input an image of a person, and a custom trained VGG-16 Deep Neural Network Model will be able to perform Face Detection and Face Emotion Classification. Specifically, we will be analyzing how downsampling, image masking techniques, and image sharpening techniques, impact model performance with accurately identifying and classifying face emotions into Happy, Sad, and Neutral. The motivation behind this project is to aid on-going research into face detection and emotion classification, but specifically focusing on physical layer augmentation techniques involving masking. With masking, we will control the number of pixels incident on the camera lens.
